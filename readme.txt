Dena analysis social media profiles and reccomends users gifts based on posts and images

Compile Requirements:
> Python 3.6 x64

Required dependency packages 

Tweepy:
pip install tweepy

Spacy:
pip install -U spacy
python -m spacy download en_core_web_lg

Textblob:
pip install -U textblob
python -m textblob.download_corpora

wikipediaapi:
pip3 install wikipedia-api

Microsoft Visual C++ 14.0 is required. Get it with "Microsoft Visual C++ Build Tools": https://visualstudio.microsoft.com/downloads/

TensorFlow (Must be below 2.0 % above 1.4.0):
pip3 install --upgrade tensorflow 

OpenCV:
pip3 install opencv-python

Keras:
pip3 install keras

ImageAI:
pip3 install imageai --upgrade

Ebay:
pip install ebaysdk

Pandas:
pip install Pandas

surprise:
pip install scikit-surprise

PySimpleGui:
pip install PySimpleGUI

resnet50_coco_best_v2.0.1.h5 required can be found in (Must be placed in Dena Folder):
https://github.com/OlafenwaMoses/ImageAI/releases/tag/1.0/
